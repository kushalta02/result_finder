# Student Result Finder

A frontend website where students can check their results by entering their details.

## Features

- Student-friendly interface
- Search results by name, roll number, and class
- Display detailed marks for each subject
- Calculate total marks and percentage
- Show pass/fail status
- Responsive design for all devices

## How to Use

1. Open `index.html` in a web browser
2. Enter the following details:
   - Student Name
   - Roll Number
   - Class (10, 11, or 12)
3. Click "Check Result" to view the results
